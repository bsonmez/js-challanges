# 77. Combinations

Given two integers n and k, return all possible combinations of k numbers out of 1 ... n.

Example 1:

```
Input: n = 4, k = 2
Output:
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]
```

### Usage

```
yarn t combine
```

Source: [leetcode.com](https://leetcode.com/problems/combinations/).
